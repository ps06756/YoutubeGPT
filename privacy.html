Privacy Policy for YouTube Video Summarizer

Last Updated: February 23, 2026

Overview

YouTube Video Summarizer is a Chrome extension that helps users summarize YouTube videos using AI. We are committed to protecting your privacy and being transparent about how we handle data.

Data Collection

This extension does NOT collect, store, or transmit any personal data to our servers. All processing happens locally in your browser or through direct API calls to your chosen AI provider.

API Key Storage

  • Your API key (Anthropic or OpenAI-compatible) is stored locally in your browser using Chrome's storage.local API
  • The key is stored only on your device and never leaves it except when making direct API calls to your chosen provider
  • We do not have access to your API key

External API Calls

  • Video transcripts are sent directly to your chosen AI provider's API (Anthropic or OpenAI-compatible)
  • No data passes through our servers
  • All API requests are made directly from your browser to the AI provider

Permissions Justification

  • storage: To save your API key and preferences locally in your browser
  • activeTab: To inject the summary sidebar on YouTube video pages
  • youtube.com host permission: To fetch video transcripts from YouTube pages
  • api.anthropic.com / OpenAI-compatible hosts: To send transcripts for summarization

Third-Party Services

This extension interacts with the following third-party services based on your configuration:

  • Anthropic API (optional, user-configured): Processes transcripts for summarization
  • OpenAI-compatible APIs (optional, user-configured): Alternative for summarization
  • YouTube: Source of video transcripts

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.
